zabbix-server部署

1、准备

　　centos7.5 系统服务器3台、 一台作为监控服务器， 两台作为被监控节点， 配置好yum源、 防火墙关闭、 各节点时钟服务同步、 各节点之间可以通过主机名互相通信。
1）所有机器关闭防火墙和selinux

2）根据架构图，实验基本设置如下：

机器名称 IP配置 服务角色 备注
server 192.168.246.228 zabbix-server 开启监控功能
node1 192.168.246.226 zabbix-agent-none1 开启
node2 192.168.246.227 zabbix-agent-none2 开启
