下面开始正式教程

本教程适用于Centos 5系-7系64位系统，经过测试，Centos 5系最适合运行，7系会出现CPU占用过高的情况

首先将

DNFServer.tar.gz
install
上传至服务器根目录

然后执行安装脚本

安装命令：

脚本详细内容如下

然后到game目录下

上传完成后运行命令：cd /root;./run

等待服务跑出‘五国’就行了

然后把客户端准备一下
四个文件夹名称
Fonts
ImagePacks2
Music
SoundPacks
一个audio.xml

然后将登录器文件夹下的四个文件拉取到客户端下
进入DNF.toml文件中，设置服务器IP地址和等级

pvf也要拉进客户端文件夹下

黑窗口里竖着显示：CN，HK，KR，MO，TW，并且下面是有至少两个感叹号的时候，这时候就代表着已经跑完“五国”了，可以登录游戏了

数据库默认帐号为：game
默认密码为：uu5!^%jg

php目录：/opt/lampp/htdocs 端口：1234

数据库目录：/opt/lampp/var/mysql

重跑命令：cd /root;./stop;./stop;./run
