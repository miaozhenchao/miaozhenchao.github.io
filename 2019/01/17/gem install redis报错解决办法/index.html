gem install redis报错解决办法

redis-cluster安装需要通过gem install redis来安装相关依赖。否则报错。
通过gem install redis执行后会出现两个问题：
1、缺少zlib依赖
问题：

解决：
通过

安装zlib库
然后集成zlib库到ruby环境

重要：ext/openssl/Makefile 增加 top_srcdir = ../..
//这一步如果不修改，make时会爆出另外一个错误

2、缺少openssl库
问题：

解决：
通过yum install openssl-devel 安装openssl库
然后集成到ruby环境
